# S.A.F.E.R.
### Smart Automated Fire Emergency Responder

S.A.F.E.R. is a compact fire-prevention robot designed to detect early signs of fire and automatically respond to stop small fires from growing. It uses flame recognition, smoke sensing, and a fast water-spray system to provide quick and effective fire suppression.
